## Interactive Demo

One cool thing about this blog is that you can **edit and run code directly** in your browser. Try it out below!

import { LiveCode } from '../../components/LiveCode'

<LiveCode code={`function Counter() {
  const [count, setCount] = useState(0)

  return (
    <div style={{ fontFamily: 'system-ui' }}>
      <h3>Interactive Counter</h3>
      <p>Count: {count}</p>
      <button
        onClick={() => setCount(c => c + 1)}
        style={{
          padding: '8px 16px',
          backgroundColor: '#3b82f6',
          color: 'white',
          border: 'none',
          borderRadius: '6px',
          cursor: 'pointer',
          marginRight: '8px'
        }}
      >
        Increment
      </button>
      <button
        onClick={() => setCount(0)}
        style={{
          padding: '8px 16px',
          backgroundColor: '#6b7280',
          color: 'white',
          border: 'none',
          borderRadius: '6px',
          cursor: 'pointer'
        }}
      >
        Reset
      </button>
    </div>
  )
}

render(<Counter />)`} noInline={true} />

Try changing the button colors or adding new features!
