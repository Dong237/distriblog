## Writing Blog Posts

### 1. Create a New Post

Create a file in `pages/posts/`:

```bash
touch pages/posts/my-awesome-post.mdx
```

### 2. Add Frontmatter

```yaml
---
title: My Awesome Post
date: 2025/1/21
description: A brief description for SEO and previews
tag: llm, tutorial, python
author: Your Name
---
```

### 3. Write Content

```markdown
# My Awesome Post

Your content here in **Markdown**...

## Subheading

- Bullet points
- Code blocks
- Images
```

### 4. Add Interactive Code (Optional)

```jsx
import { LiveCode } from '../../components/LiveCode'

<LiveCode code={`function hello() {
  return "Hello, World!"
}`} />
```

For components needing `render()`:

```jsx
<LiveCode code={`function Counter() {
  const [count, setCount] = useState(0)
  return <button onClick={() => setCount(c => c + 1)}>Count: {count}</button>
}
render(<Counter />)`} noInline={true} />
```

### 5. Deploy

```bash
git add .
git commit -m "Add post: My Awesome Post"
git push
```

Vercel auto-deploys in ~30 seconds.

## Design System

### Colors

| Mode | Background | Text |
|------|------------|------|
| Light | `#FAF7F2` (cream) | `#1a1a1a` (charcoal) |
| Dark | `#1a1816` (warm dark) | `#f5f5f4` (light) |

### Typography

- **Headings:** Georgia (serif)
- **Body:** Inter (sans-serif)
- **Code:** JetBrains Mono / system monospace

### Layout

- Content width: 720px
- Sticky header with blur
- Mobile responsive

---

## Tech Stack

- **Framework:** Next.js 14 + React 18
- **Blog Engine:** Nextra 3
